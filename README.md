# Privacy-First DEX
### A decentralized exchange for India. No KYC. Full control.

---

## Why Privacy-First DEX?

**95% of Indian traders use centralized exchanges** — giving up control, privacy, and dealing with KYC restrictions.

We're building the alternative: **a fully decentralized, privacy-preserving exchange where you own your assets and your data.**

---

## What We're Building

- **Zero KYC**: Trade without providing personal information
- **Self-Custody**: Your private keys, your crypto — always
- **Multi-Chain**: Launch on Polygon + BSC for low fees and speed
- **Privacy by Default**: Advanced protocols to keep trades private
- **Open Source**: Transparent, auditable, community-driven

---

## Tech Stack

- **Smart Contracts**: Solidity 0.8.20 (Polygon + BSC)
- **Frontend**: React + Vite (coming soon)
- **Testing**: Hardhat + Mocha + Ethers.js
- **Networks**: Polygon Amoy (testnet) → Polygon Mainnet + BSC (launch)

## Development Roadmap

## Getting Started

### Prerequisites
- Node.js 18+
- MetaMask wallet with Mumbai testnet configured

### Setup

```bash
# Install dependencies
npm install

# Compile contracts
npx hardhat compile

# Run tests
npx hardhat test

# Deploy to Mumbai testnet (coming soon)
npx hardhat run scripts/deploy.ts --network mumbai
```
